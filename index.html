blockchain2graph

blockchain2graph extract data from the bitcoin blockchain into a graph database (neo4j for now). Once the data is there you can do anything you want like :

  • Process and query blockchain data with the neo4j query language.
  • Build web sites that presents information like Blockchain info.
  • Add meta information to addresses and transactions directly in the nodes.

At Blockchain Inspector, we are using Artificial Intelligence to fight fraud in the Blockchain. To do so, we needed to have a more conveniant way to represent blockchain data than the one offered by bitcoind. Thanks to this tool, we can analyse blockchain data as transactions between nodes and add meta informations on them.
